Problem

ให้ทุกคนลองคิดว่าหน้าเว็บต่อไปนี้ควรวางโครงสร้าง HTML อย่างไร และ ใช้การจัดรูปแบบ grid หรือ flex อย่างไร
ให้ทุกคนหา

  • เว็บ ข่าว มา 1 เว็บ ไป
  • เว็บ ช๊อปปิ้งมา 1 เว็บ ไป
  • เว็บ กระทู้ หรือ การเรียนการสอนมา 1 เว็บ ไป

ให้ส่ง link เว็บและเขียนสรุปคราวๆว่าจะว่างโครงสร้างในการเขียนเว็บอย่างไร
ถ้าสามารถทำได้อยากให้ ทำลงในไฟล์ HTML คล้ายเว็บตัวอย่างมาจะยิ่งดี

เนื้อหาข่าว


ในส่วนเนื้อหาข่าว ผมยังทำลิงค์ไว้ให้คลิกไปชมหน้าอื่นๆได้อีก คือ หน้าหมวดหมู่ข่าว ซึ่งในที่นี้เป็นข่าววิทยาศาสตร์
และเข้าไปดูหน้าแรกของเว็บไซต์ได้ ซึ่งผมวางโครงสร้างหน้าเว็บไว้แบบเดียวกันด้วย flex-container
แบ่งเป็น 3 column โดยใช้ flex-grow กำหนดให้ column กลางมีกว้างมากที่สุด ส่วนซ้ายขวา กำหนดไว้
เพื่อเป็นระยะขอบของหน้าเว็บเท่านั้น
