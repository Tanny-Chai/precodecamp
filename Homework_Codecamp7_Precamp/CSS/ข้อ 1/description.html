คำอธิบาย



  • Tooltips: คือ แถบข้อความที่จะปรากฏขึ้นเมื่อเราเอาเมาส์ไปวางไว้บน object ที่ถูกกำหนดค่าไว้
  • Pagination คือ แถบ button ให้ user สามารถคลิกเมาส์เลือกไปดูในหน้าเว็บต่างๆที่มีได้ มักถูกจัดวางไว้
    ด้านล่างของหน้าเว็บ
  • Navigation Bar คือ แถบแสดงรายการหัวข้อของเว็บ ให้ user สามารถคลิกเลือกไปในหัวข้อหน้าที่ต้องการได้ มักถูกจัดวางไว้เป็นแถบแนวนอนขอบบน หรือ ไว้เป็นแถบแนวตั้ง
  • Dropdown คือ แถบแสดงรายการหที่จะปรากฏขึ้นเมื่อนำเมาส์ไปวางที่ button ให้ user สามารถคลิกเลือกไปในรายการที่ต้องการได้
หน้าต่อๆไปจะเป็นการแสดงตัวอย่างของแต่ละเครื่องมือ ดังต่อไปนี้
